# **OHS 11g**

---



<br/>

## **.apachectl 권한 변경**

---



**파일 위치**

```shell
${ORACLE_HOME}/ohs/bin
```

<br/>



**권한 변경**

```shell
chown root .apachectl
chmod 6750 .apachectl
```

→ <mark style="background-color: #fff5b1">apachectl 앞의 . 꼭 붙일 것!</mark>

# **OHS 12cR1**

---

<br/>



## **1. hasbind 권한 변경**

---



**파일 위치**

```shell
${ORACLE_HOME}/oracle_common/bin
```

<br/>

**권한 변경**

```shell
chown root hasbind
chmod 4756 hasbind
```





<br/>

## **2. /etc/cap.ora 파일 생성**

---

**cap.ora 파일 복사**

````shell
cp /tmp/cap.ora /etc/cap.ora
````

<br/>

**권한 변경**

````shell
chmod 644 /etc/cap.ora
````



<br/>

## **3. /tmp/cap.ora 파일 삭제**

---



**파일 삭제**

````shell
rm /tmp/cap.ora
````

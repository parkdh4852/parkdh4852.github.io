# **풀이**

---

**`완전 탐색`**을 이용해 풀 수 있습니다.

**9명의 난쟁이들의 키 합 - 2명의 키 = 100** 를 찾는 방법으로 접근했습니다.



<br/>

# **소스 코드**

---

```c++
#include <iostream>
#include <algorithm>
using namespace std;

#define N 9

int main(int argc, char**argv) {
	
	ios::sync_with_stdio(0);
	cin.tie(0);
	
	int dwarf[N], sum=0;
	bool flag = false;
	
	for(int i=0; i<N; i++){
		cin >> dwarf[i];
		sum+=dwarf[i];
	}
	
	
	for(int i=0; i<N-1; i++){
		sum-=dwarf[i];
		
		for(int j=i+1; j<N; j++){
			sum-=dwarf[j];
			
			if(sum==100){
				dwarf[i]=0;
				dwarf[j]=0;
				flag = true;
				break;
			}
			sum+=dwarf[j];
		}
		sum+=dwarf[i];
		if(flag)
			break;
	}
	
	sort(dwarf, dwarf+N);
	
	for(int i=2; i<N; i++){
		cout << dwarf[i] <<" ";
	}
	
	
	return 0;
}
```

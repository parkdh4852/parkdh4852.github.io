# **풀이**

---

to_string 함수를 이용하여 숫자를 문자열로 변환할 수 있습니다.

**`int to string`**



<br/>

# **소스 코드**

---



```c++
#include <string>
#include <vector>

using namespace std;

string solution(vector<string> seoul) {
    string answer = "";
    
    for(int i=0; i<seoul.size(); i++){
        if(seoul[i]=="Kim"){
            answer= "김서방은 "+to_string(i)+"에 있다";
            break;
        }
    }
    
    return answer;
}
```

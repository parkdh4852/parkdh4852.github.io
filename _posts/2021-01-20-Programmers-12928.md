# **풀이**

---

1~n까지 돌며 나누어 떨어지는 수들을 answer에 더해줍니다.



<br/>

# **소스 코드**

---



```c++
#include <string>

using namespace std;

int solution(int n) {
    int answer = 0;
    for(int i=1; i<=n; i++){
        if(n%i==0)
            answer+=i;
    }
    return answer;
}
```

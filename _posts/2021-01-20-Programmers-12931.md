# **풀이**

---

n을 10으로 나눈 **나머지 값**으로 **`맨 끝자리(일의 자리)`**를 구할 수 있습니다.

n을 10으로 나눈 **몫**을 구하면 맨 끝자리를 제외한 수를 남기게 됩니다.

n이 0이 되기전까지 이를 반복해주면 됩니다.

<br/>

> ex) n = 123
>
> n%10 = 3
>
> n/10 = 12
>
> ...(반복)..



<br/>

# **소스 코드**

---



```c++
using namespace std;
int solution(int n)
{
    int answer = 0;

    while(n>0){
        answer += n%10;
        n = n/10;
    }

    return answer;
}
```

# **풀이**

---



자릿수 더하기 문제의 연장선입니다. (앞 포스팅 참고)

<br/>

**`하샤드 수`**는 각 자릿수 숫자의 합으로 그 수가 나누어지는 양의 정수를 의미합니다.

모든 자릿수의 합으로 그 수가 나누어 떨어지면 true, 아니면 false를 출력합니다.

<br/>

# **소스 코드**

---



```c++
#include <string>

using namespace std;

bool solution(int x) {
    int sum = 0;
    int num = x;
    
    while(num>0){
        sum += num%10;
        num = num/10;
    }
    
    return x%sum==0 ? true : false;
    
}
```

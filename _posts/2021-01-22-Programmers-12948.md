# **풀이**

---



아주 간단히 풀 수 있는 문제입니다.

전화번호의 뒷 4자리를 뺀 나머지를 *로 바꿔주면 됩니다.

<br/>

# **소스 코드**

---



```c++
#include <string>

using namespace std;

string solution(string phone_number) {
    
    for(int i=0; i<phone_number.size()-4; i++){
        phone_number[i] = '*';
    }
    
    return phone_number;
}
```

# **문제**

---



문제를 보시려면 [링크](https://www.acmicpc.net/problem/1085)를 클릭해주세요. 

<br/>

# **풀이**

---

**현수 위치(x,y)**에서 아래 꼭짓점(0,0)과의 거리**`(x-0, y-0)`**와  위 꼭짓점(w,h) 과의 거리**`(w-x, h-y)`**, 즉 4개의 값 중에서 최솟값을 구해줍니다.



<br/>

# **소스 코드**

---



```c++
#include <iostream>

using namespace std;

int main(){
	
	ios::sync_with_stdio(0);
	cin.tie(0);
	
	int x,y,w,h;
	
	cin >> x >> y >> w >> h;
	
	int min=w-x;

	if(h-y<min) min=h-y;
		
	if(x<min) min=x;
		
	if(y<min) min=y;
		
	cout << min;	
	
	return 0; 

}
```
